# 3D Knight's Tour — 立体ナイトツアー

ナイト（騎士）の動きで全マスをちょうど 1 回ずつ通る「ナイトツアー」を、
**3 次元の盤面**で挑戦できるパズルです。Three.js による 3D 表示で、視点を回転しながらプレイできます。

> ※ リポジトリ名は `3d-kngiht-tour` ですが、正式には *3D Knight's Tour* です（綴りの typo）。

## 🔗 リンク

- リポジトリ: https://github.com/Sora3141/3d-kngiht-tour
- デモ: https://sora3141.github.io/3d-kngiht-tour/

## 概要

- 3 次元グリッド上のナイトツアー。平面より直感が効かず難易度が高いのが特徴。
- 移動可能なマスをハイライトし、クリック / タップで進む。
- **UNDO**（1 手戻る）・**REBOOT**（やり直し）対応。
- ドラッグで視点回転・ホイール / ピンチでズーム。

## 使用技術

- **Three.js**（3D 描画）
- HTML / CSS / Vanilla JavaScript
